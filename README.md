# AutoDangKiTin_TLU

Tool đăng kí học (chắc chắn xịn hơn web của nhà trường) với sự hỗ trợ 67% từ Gemini 3 Pro

Tính năng chính:
- Đăng kí tín chỉ với 3667 cách đấm request
- Đồng bộ lịch học lên google Lịch
- Tự động đăng kí tín chỉ theo lựa chọn có sẵn
